# Azure SOC Lab

## Overview
Hands-on security operations lab simulating real-world attacks 
in Microsoft Azure and detecting them using Microsoft Sentinel and KQL.

## Tools Used
- Microsoft Azure / Entra ID
- Microsoft Sentinel
- KQL (Kusto Query Language)
- Microsoft Defender Portal
- Network Watcher

## Labs Completed
- Exercise A: Password Spray Attack Simulation & Detection
- Exercise B: Privilege Escalation Detection
- Exercise C: NSG Misconfiguration Analysis & Remediation

## Key Skills Demonstrated
- Threat detection using KQL
- Incident response workflows
- Identity security monitoring
- Network security configuration
